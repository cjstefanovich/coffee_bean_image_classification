# Coffee Bean Roast Level Classification

This project classifies images of individual coffee beans into four roast levels (**Green**, **Light**, **Medium**, **Dark**) using image processing and a Random Forest classifier. It supports quality control by evaluating classification accuracy on held-out test data, on custom images, and by assessing batch-level roast consistency.

---

## How to Run the Project

All code is in the `Code/` folder. Run the following in order. Execute notebooks from the `Code/` directory so that paths to `coffee_data/` and saved model files resolve correctly.

### 1. Get the dataset

- Download the **Coffee Bean Dataset Resized (224 x 224)** from Kaggle:  
**[https://www.kaggle.com/datasets/gpiosenka/coffee-bean-dataset-resized-224-x-224/data](https://www.kaggle.com/datasets/gpiosenka/coffee-bean-dataset-resized-224-x-224/data)**
- The dataset will be downloaded as a compressed file named `archive.zip`. Extract it somewhere locally.
- Move the dataset contents into the empty `original_dataset` folder so that under `Code/` you have:
  - `Code/coffee_data/original_dataset/train/` with subfolders `Green/`, `Light/`, `Medium/`, `Dark/`
  - `Code/coffee_data/original_dataset/test/` with the same four subfolders
  - (Optional: the CSV `Coffee Bean.csv` that comes with the dataset can be used for reference; the notebooks build paths from the folder structure.)

This folder structure is how the dataset comes by default when you download it.

### 2. Preprocessing and EDA - `coffee_preprocessing_EDA.ipynb`

- **What it does:** Merges train and test images into single per-roast folders (`Green/`, `Light/`, `Medium/`, `Dark/` under `coffee_data/`), finds and removes duplicate images (by file hash), runs background removal on every image and overwrites them in place, and performs exploratory data analysis.
- **Run:** Open and run all cells. Ensure `coffee_data/original_dataset/` exists (see step 1). When finished, `coffee_data/Green`, `coffee_data/Light`, etc. contain the processed images used for training.

### 3. Train the model - `coffee_ML_model.ipynb`

- **What it does:** Loads the processed images from `coffee_data/`, extracts 15 color/texture features per image, splits data into train/test (stratified 80/20), runs hyperparameter search (RandomizedSearchCV) for the Random Forest, trains the best model, evaluates on the test set, and saves the model artifacts in `Code/`.
- **Run:** Open and run all cells. Requires the preprocessing notebook to have been run first. Produces:
  - `Code/coffee_rf_model.joblib`
  - `Code/coffee_scaler.joblib`
  - `Code/coffee_metadata.joblib`

### 4. Evaluation - `coffee_model_eval.ipynb`

- **What it does:**
  - **Custom images:** Loads images from `coffee_data/custom_test_images/` and their labels from `custom_image_labels.csv`, preprocesses them (resize 224×224, background removal), runs the saved model, and reports accuracy, confusion matrix, and per-class metrics.
  - **Unlabeled images:** Predicts roast level for any image in `custom_test_images/` that is not listed in the CSV.
  - **Batch roast consistency:** Builds synthetic “perfect”, “good”, and “bad” batches from the roast-level folders, runs the model on each batch, and reports the percent of beans predicted as the batch’s intended roast (consistency).
- **Run:** Open and run all cells. Requires the ML notebook to have been run so that the three `.joblib` files exist in `Code/`. 
  - Optional: add your own images to the folder `coffee_data/custom_test_images/` and write image file names and true labels to `custom_image_labels.csv` (columns: `image_filename`, `label`) for custom evaluation. Images with true labels will have accuracy/metrics calculated for them. If there is no true label for an image, it will only receive a predicted label.

## Dependencies and Environment

**Python:** 3.x
**Libraries:**


| Package       | Use                                                                                                                                                                                                 |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| numpy         | Arrays and numerical operations (all notebooks + coffee_utils)                                                                                                                                      |
| opencv-python | Image I/O, resize, color conversion RGB/HSV, morphology, thresholding (coffee_utils, preprocessing, eval)                                                                                           |
| scipy         | `binary_fill_holes` for mask cleanup (coffee_utils)                                                                                                                                                 |
| pandas        | DataFrames for paths, labels, results (preprocessing, ML model, eval)                                                                                                                               |
| matplotlib    | Plots: confusion matrix, ROC, learning curve, batch consistency, EDA (preprocessing, ML model, eval)                                                                                                |
| scikit-learn  | RandomForestClassifier, LinearRegression, StandardScaler, LabelEncoder, train_test_split, RandomizedSearchCV, Pipeline, metrics, calibration, cross_val_score, learning_curve, plot_tree (ML model) |
| joblib        | Save/load model, scaler, metadata (ML model, eval)                                                                                                                                                  |
| pathlib       | Path handling (all notebooks)                                                                                                                                                                       |
| shutil        | Copying files when merging train/test (preprocessing)                                                                                                                                               |
| re            | Regex for parsing filenames when renaming test images (preprocessing)                                                                                                                               |
| hashlib       | MD5 hashing to detect duplicate images (preprocessing)                                                                                                                                              |
| collections   | defaultdict, Counter for confusion counts and batch distributions (ML model, eval)                                                                                                                  |
| warnings      | Suppressing warnings (all notebooks)                                                                                                                                                                |


Install third-party packages you don't have with:

```bash
pip install numpy opencv-python scipy pandas matplotlib scikit-learn joblib
```
